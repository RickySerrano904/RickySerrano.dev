# Shipping a tiny design system

This placeholder post captures the first iteration of a small design system. It
is not exhaustive, but it is consistent enough to ship real work.

## What shipped

- Typography scale with two fonts and clear hierarchy.
- A color palette with one accent and flexible neutrals.
- Three core components: buttons, cards, and tags.

## Tokens first

Tokens create shared language before components exist. I start with spacing,
type, and color, then wire them into CSS variables.

## Components second

With tokens in place, components become an assembly exercise. This keeps the
system small and consistent.

## Next steps

Add forms, navigation variants, and content modules as the product grows.
